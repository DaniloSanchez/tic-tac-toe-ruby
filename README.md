Tic Tac Toe
=========

This program was made in ruby.

They have 5 classes,

 - main
 - dashboard
 - game
 - valitador
 - display

To run the game you need to have ruby on the computer. You only need to run the main.rb

Run
--------------

```sh
ruby main.rb
```
